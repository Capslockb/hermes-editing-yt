hermes-editing-yt

Subtitle-Driven Auto Video Editor — Your videos, editable by text.

One MCP call. Local GPU. Free forever. No API keys.

hermes-editing-yt — raw docs

This directory is the single source of truth for the project's written documentation. Pages here are Markdown-first; the generated site in ../docs-site/ is produced by ../scripts/build_docs_site.py.

The rich public website lives in ../site/ and deploys separately to https://capslockb.github.io/hermes-editing-yt/.

Status legend

Every capability in this tree is tagged with one of these labels:

Label Meaning
Working Code exists in the repo and has an executable verification path.
Partial Code exists but depends on a local service, credential, or binary that the user must provide.
Planned Roadmapped design; implementation not yet in main.
Research Mentioned only as a future integration target until code + tests land.
